## Опис проєкту
Це веб-додаток, що реалізує чат-підтримку з штучним інтелектом.  
Бот відповідає на повідомлення користувача за допомогою OpenAI API та зберігає історію діалогу у Firebase Firestore.

---

## Технології
- Node.js
- Express.js
- HTML/CSS/JS
- JWT авторизація
- OpenAI API
- Firebase Firestore
- Git / GitHub

---

##  Можливості системи:
- Реєстрація і авторизація користувача
- Live чат між користувачем та ботом
- Збереження історії повідомлень
- Перегляд історії у особистому кабінеті
- Локальний запуск сервера
